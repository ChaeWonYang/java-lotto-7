# 기능 목록

### 1. 구입 금액 입력
    [] 구입 금액 예외 처리
    [] 구입 금액을 전달 

### 2. 로또 구입 갯수 계산
    [] 전달 받은 구입 금액으로 로또 구입 갯수 계산

### 3. 로또 구입 갯수 출력
    [] 로또 구입 갯수 출력 

### 4. 로또 구입 갯수 전달
    [] 로또 구입 갯수 전송 

### 4. 로또 번호 뽑기
    [] 전송 받은 로또 구입 갯수를 바탕으로 로또 번호 뽑기
    [] 뽑은 로또 번호 저장 

### 5. 뽑힌 로또 번호 출력
    [] 뽑힌 로또 번호 출력

### 6. 당첨 번호 입력
    [] 당첨 번호 입력받고 전송 

### 7. 보너스 번호 입력
    [] 보너스 번호 입력받고 전송 

### 8. 당첨 유형 계산
    [] 당첨 유형 계산 및 저장 

### 9. 당첨 통계 계산
    [] 당첨 통계 계산 및 저장 

### 10. 당첨 통계 출력
    [] 당첨 통계 출력
